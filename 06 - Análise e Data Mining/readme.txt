> O arquivo main.py executa o programa.

> O arquivo ConheceNormalizada contém a classe que executa os processos relacionados a view criada no banco de dados.

> O arquivo graficos.py contém a classe que executa os processos relacionados aos gráficos criados com os dados do banco.

> O arquivo primeiras.py contém a classe que executa os processos relacionados a Rating dos filmes e artistas musicais.

> Os arquivos SampleApp.py e StartPage.py são respectivamente as classes que iniciam a aplicação e mostram a tela inicial do programa.

> O arquivo extras.py contém a classe que executa os processos relacionados a mais duas funções da aplicação que achamos úteis.
                                                                                   
> O arquivo __init__.py serve para que as classes sejam carregadas do diretório onde elas estão, sem precisar adicionar o caminho.

OBS.: Para a aplicação ser executada é necessário ter instalado a biblioteca psycopg2. Ela foi feita para Python 2.7. Se possui Python 3, troque "import Tkinter" nas classes por "import tkinter".
